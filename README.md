Kraken Blog Example
===========

```
git clone git@github.com:zaak/krakenjs-blog-example.git
cd krakenjs-blog-example
npm install
bower install
```

Before you start remember to configure the database connection [here](https://github.com/zaak/krakenjs-blog-example/blob/master/config/config.json#L11).
Sequelize will create required tables.

Then just:
```
npm start
```
